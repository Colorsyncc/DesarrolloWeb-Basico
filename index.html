Creando nuestro primer proyecto

Una vez descargados e instalados los programas procederemos a abrir Visual studio code, les dejaré mi configuracion personal para que puedan trabajar de manera mas comoda. Lo que tienen que hacer es lo siguiente: Archivo -> Preferencias -> Configuración
Una vez que nuestro archivo este modificado como esta en la imagen vamos a hacer lo siguiente:

  • 1.-Crear nueva carpeta y darle un nombre, en la ubicación que desees.
  • 2.-Abrir Visual Studio Code y dirigirse a Archivo -> Abrir carpeta...
  • 3.-Buscan su carpeta recién creada, la seleccionan y le dan click a "Seleccionar carpeta".
  • 4.-Puedes ver el ejemplo haciendo click aquÍ .
